Introduction

The Nathan’s Famous Fourth of July International Hot Dog Eating Contest has a 100+ year tradition. Over time, competitors have pushed the limits of human capacity in ways no one originally imagined.

This module explains the process of creating effective data visualization plots using tidyverse and the closeread HTML extension.

Closeread is a custom format for Quarto that enables scrollytelling features for HTML documents.

Learning Objectives

By the end of this activity, you will be able to:

  1. Create effective data visualization plots from the data through contextual analysis.

  2. Analyze and describe the trends and significant features of the data set with appropriate annotations.

  3. Use the closeread HTML extension for Quarto documents to create a scrollytelling file.

  4. Interpret general trend information of a specific data set from the visualization plots and annotations.

Methods

Technology requirement: The activity handout provides a guidance Quarto document and a solved HTML, for reference on solutions and code references.

  1. Plots: Familiarity with tidyverse, mainly ggplot2, dplyr and tidyr as a techniques for statistical analysis and data representation is crucial. You should understand how to use RMarkdown code to perform changes on the data set.

  2. Basic understanding of the closeread syntax to create sections, stickies and reference files within the code to appear as desired in the final HTML document.

Data

A data frame for 69 champions of the Nathan’s Famous Hot Dog eating competition from 1972 until 2025. The data has been scraped from the web source Wikipedia, “Nathan’s Famous Hot Dog Eating Contest”, and cleaned with the R tidyverse library. The variable for the dogs eaten is the result of calculating the ratio of the hot dogs each winner ate in non-extra time periods by the time of the competition in minutes. This was done because the competition times are not the same for all the years in the data set.

Download data: hotdog_clean.csv

Variable Descriptions
