Let’s get started

To begin, you will need to set up Closeread. Copy and paste the following block of code into a new qmd file.

---
title: My First Closeread
author: Your Name
format: closeread-html
---

Get the packages we need to work with our data (uncomment this chunk)
#```{r}
library(readr) # This will help us read in our data
library(tidyverse) # This will help us work with our data and make plots
#```

Get our data (uncomment this chunk)
#```{r}
hotdog_clean <- read_csv("hotdog_clean.csv")
#```

:::{#cr-intro}

This is where you will start working! The three colon symbols denote a section, which is how your closeread will be organized.

:::

Next, you will need to install the Closeread extension. At the bottom of your screen, you should have a “console” open where the output of your code appears as you run it in RStudio. Next to console, you’ll see something called the “Terminal”, open that up and paste in the following code:

quarto add qmd-lab/closeread

Now, separate the data into hotdog_men a dataset of only the male winners, and hotdog_women a dataset of only the women who have won.

CautionReveal the Code 
hotdog_men <- hotdog_clean |> filter(Sex == 1)
hotdog_women <- hotdog_clean |> filter(Sex == 0)
